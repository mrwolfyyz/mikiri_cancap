Setup & Configuration

Step 1 (Optional): Configure Google Shared Drive

Reports are always viewable in your browser. To also save them automatically to a Google Shared Drive folder, follow these steps:

⚠️ IMPORTANT: You must use a Shared Drive (Team Drive), not a personal "My Drive" folder. The service account only has access to Shared Drives.
1.1 Create or choose a Shared Drive folder
  1. Go to Google Drive: Visit drive.google.com
  2. Navigate to Shared Drives
    • In the left sidebar, click Shared drives
    • If you don't see this option, you may need a Google Workspace account
  3. Create or choose a folder inside a Shared Drive:
    • Open an existing Shared Drive
    • Click + New → New folder
    • Name it something like "{{folderName}}" or "{{folderAltName}}"
  4. Open the folder by double-clicking it
  5. Copy the URL from your browser's address bar
    • It should look like: https://drive.google.com/drive/folders/1abc...
    • The long string after /folders/ is your folder ID
⚠️ Personal Drive won't work! The folder MUST be inside a Shared Drive. Personal "My Drive" folders are not accessible to the service account.
1.2 Share the folder with the service account (CRITICAL)
⚠️ IMPORTANT: Without this step, reports will not be uploaded to Drive (you can still view them in your browser).
  1. Right-click the folder in Google Drive (the folder you created in the Shared Drive)
  2. Click Share (or click the share icon)
  3. In the "Add people and groups" field, paste this email:
    Loading...
  4. Set the permission to Content manager or Manager
    • Don't use "Viewer" or "Commenter" - it needs write access!
  5. Uncheck "Notify people" (optional - it's just a service account)
  6. Click Done
💡 What's this? This is the service account that runs the backend. Giving it access lets it automatically upload reports to your Shared Drive folder.

Step 2: Set Up Obsidian (Optional)

{{#if origination}} Obsidian gives you a powerful way to view and link between loan origination reports. It's free and works great with Shared Drives.

🔍 Used by loan officers: Obsidian is a powerful tool for organizing and reviewing borrower intelligence reports.
{{/if origination}} {{#if skiptrace}} Obsidian gives you a powerful way to view and link between skip trace reports. It's free and works great with Shared Drives.
🔍 Used by investigators: Obsidian is the go-to tool for investigative journalists, intelligence analysts, and skip tracers working complex cases.
{{/if skiptrace}}

2.1 Download & Install Obsidian
  1. Visit obsidian.md
  2. Click Download for your operating system (Windows/Mac/Linux)
  3. Install the app (it's completely free - no account needed)
2.2 Install Google Drive Desktop
  1. Visit google.com/drive/download
  2. Download Drive for desktop (not Backup and Sync)
  3. Install and sign in with your Google account
  4. After installation, Shared Drives will appear in your file explorer:
    • Mac: /Volumes/GoogleDrive/Shared drives/YourSharedDrive
    • Windows: G:\Shared drives\YourSharedDrive
    • Linux: Follow Google's setup for your distro
💡 Tip: Drive Desktop keeps files synced in real-time. Changes appear instantly!
2.3 Configure Obsidian Vault
  1. Open Obsidian
  2. Click Open folder as vault (or create a new vault)
  3. Navigate to your Shared Drive folder:
    • Mac: /Volumes/GoogleDrive/Shared drives/YourSharedDrive/{{folderName}}
    • Windows: G:\Shared drives\YourSharedDrive\{{folderName}}
  4. Click Open
  5. When you run investigations, the markdown files will appear automatically in Obsidian!
⚠️ Important: The folder path must be inside a synced Shared Drive for this to work.

Troubleshooting

Reports aren't appearing in my Drive folder

Check these common issues:

  1. Are you using a Shared Drive?
    • Personal "My Drive" folders won't work!
    • The folder MUST be inside a Shared Drive (Team Drive)
    • Check that the folder is in the "Shared drives" section, not "My Drive"
  2. Did you share the folder with the service account?
    • Go back to Step 1.2 above - this is the #1 reason reports don't appear!
    • Make sure functions-sa@YOUR_PROJECT_ID.iam.gserviceaccount.com has "Content manager" or "Manager" access
    • Right-click the folder → "Share" → Check "Manage access" to verify
  3. Verify your folder URL is correct
    • Go back to Step 1.1 and double-check the URL
    • Make sure you copied the entire URL including the long folder ID
    • Try clicking "{{viewReportText}}" after a completed investigation
  4. Wait a moment
    • Drive can take 5-10 seconds to show new files
    • Try refreshing your Drive browser tab
  5. Check investigation completed successfully
    • Did you see "{{successMessage}}" message?
    • If investigation failed, reports won't be generated
Still not working? Email brad@bbak.ai with your job ID (shown during investigation)
Files not syncing to Obsidian

Try these fixes:

  1. Verify Google Drive Desktop is running
    • Look for the Drive icon in your system tray (Windows) or menu bar (Mac)
    • Click it and make sure it says "Synced" or "Up to date"
    • If it's paused, click Resume
  2. Check Obsidian is pointed to the right folder
    • In Obsidian, look at Settings → Files & Links
    • Verify the vault path is inside a synced Shared Drive
    • Example: /Volumes/GoogleDrive/Shared drives/YourSharedDrive/{{folderName}}
  3. Force a refresh
    • In Obsidian, press Cmd+R (Mac) or Ctrl+R (Windows)
    • Or close and reopen Obsidian
  4. Check sync status in Drive Desktop
    • Right-click the Drive icon → Preferences
    • Make sure your Shared Drive is enabled for syncing
Investigation is taking too long

Normal timing:

  • Most investigations: 60-90 seconds
  • Complex cases: Up to 2 minutes

If it's stuck for 3+ minutes:

  1. Check your internet connection
  2. Try refreshing the page and submitting again
  3. If it keeps failing, email brad@bbak.ai with the job ID
💡 Tip: The job ID is shown during investigation - screenshot it if reporting issues!
Error: "Invalid Drive folder URL"

Your URL must look like one of these:

  • https://drive.google.com/drive/folders/1abc...
  • https://drive.google.com/drive/u/0/folders/1abc...

Common mistakes:

  • ❌ Pasting a file URL instead of folder URL
  • ❌ Pasting a shared link that has extra parameters
  • ❌ Missing the folder ID (the long string after /folders/)
  • ❌ Using a personal "My Drive" folder instead of a Shared Drive

Fix:

  1. Go back to Step 1.1 above
  2. Make sure you're in a Shared Drive, not "My Drive"
  3. Make sure you're copying from the address bar (not a share link)
  4. Make sure you've opened the folder first
I don't have access to Shared Drives
⚠️ Shared Drives require Google Workspace

Shared Drives (formerly Team Drives) are only available with:

  • Google Workspace (Business Starter, Business Standard, Business Plus, Enterprise)
  • Google Workspace for Education
  • Google Workspace for Nonprofits

If you only have a personal Gmail account:

  • You'll need to upgrade to Google Workspace (~$6-18/user/month)
  • Or ask your IT admin to create a Shared Drive for you
  • Visit workspace.google.com to upgrade
